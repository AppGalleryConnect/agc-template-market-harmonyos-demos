改签须知

改签规则

以下是关于航班改签的相关规定，请在改签前仔细阅读：

  • 航班改签仅限于在原航班起飞前进行，起飞后不予改签。
  • 改签需支付相应的改签费用，具体费用请根据航空公司规定及票种确认。
  • 改签后的航班可能存在票价差异，需按差额补足。
  • 改签需提供有效的身份证件及相关证明材料。

改签流程

  1. 进入航空公司官网或APP，选择“改签”功能。
  2. 输入您的航班信息并核对原航班信息。
  3. 选择您希望改签的航班，并确认航班余位及票价差。
  4. 支付改签费用，并收到新的电子登机牌。
  5. 改签完成后，请及时确认新的航班信息。

注意事项

重要提示：在改签过程中，请确保您提供的个人信息正确无误，并及时完成支付，避免影响您的行程安排。
  • 如果改签的航班发生变化，可能会影响您的行程，请提前做好时间安排。
  • 部分特殊票种可能无法进行改签，请根据您的票种与航空公司确认。

其他常见问题

Q: 如果我错过了航班，可以改签吗？
A: 如果您错过了航班，可能无法进行改签，建议尽早与航空公司联系处理。
Q: 改签后是否可以退票？
A: 改签后根据航空公司政策可能无法退票，请提前了解相关政策。
